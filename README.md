# 🔥 Streak Tracker

A premium, minimalist Android application built with **Expo SDK 54** and **React Native** to track your habits, milestones, and personal records with style.

![Streak Tracker Mockup](assets/mockup.png)

## ✨ Features

- 📊 **Dynamic Dashboard**: Beautifully designed cards to track multiple streaks at a glance.
- 📅 **Interactive Calendar**: Visualize your progress with a custom streak-colored calendar view.
- 📈 **Detailed Stats**: Deep dive into your streak history, best records, and total activity.
- 📱 **Android Widgets**: Keep your most important streaks right on your home screen.
- 🔒 **Privacy First**: All data is stored locally on your device using SQLite.
- ⚡ **High Performance**: Built with Drizzle ORM and Zustand for a snappy, responsive experience.
- 🎨 **Premium UI**: Crafted with NativeWind (Tailwind CSS) for a modern, fluid aesthetic.

## 🛠️ Tech Stack

- **Framework**: [Expo](https://expo.dev/) (SDK 54) & [React Native](https://reactnative.dev/)
- **Navigation**: [Expo Router v3](https://docs.expo.dev/router/introduction/) (File-based routing)
- **Database**: [Drizzle ORM](https://orm.drizzle.team/) with [Expo SQLite](https://docs.expo.dev/versions/latest/sdk/sqlite/)
- **State Management**: [Zustand](https://github.com/pmndrs/zustand)
- **Styling**: [NativeWind v4](https://www.nativewind.dev/) (Tailwind CSS for React Native)
- **Icons**: [Lucide React Native](https://lucide.dev/guide/packages/lucide-react-native)
- **Animations**: [React Native Reanimated](https://docs.swmansion.com/react-native-reanimated/)

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or later)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)
- [Expo Go](https://expo.dev/client) (for development) or an Android Emulator

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Sanju2op/streak-tracker.git
   cd streak-tracker
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run android
   ```
